# Fyne 核心类型速查参考

## 1. CanvasObject — 可绘制对象接口

**所有 UI 元素的根接口**。约定了位置、大小、可见性的基本行为。

```go
type CanvasObject interface {
    MinSize() Size
    Move(Position)
    Position() Position
    Resize(Size)
    Size() Size
    Hide()
    Show()
    Visible() bool
    Refresh()   // 对象内部状态改变需重绘时调用
}
```

**重要规则**：
- 若对象由 Layout 管理，**不应手动调用** `Move()` 和 `Resize()`
- `Refresh()` 用于通知对象状态变更需要重绘
- 所有 Canvas 上的原始图形（`canvas.Rectangle`, `canvas.Text`, `canvas.Image` 等）都实现了此接口

## 2. Widget 体系

### Widget 接口

```go
type Widget interface {
    CanvasObject
    CreateRenderer() WidgetRenderer
}
```

### WidgetRenderer 接口

```go
type WidgetRenderer interface {
    Destroy()
    Layout(Size)
    MinSize() Size
    Objects() []CanvasObject
    Refresh()
}
```

### BaseWidget — Widget 基类

位于 `fyne.io/fyne/v2/widget` 包，提供位置、大小、可见性和重绘的标准实现。

**嵌入 BaseWidget 的自定义 Widget 模板**：

```go
type MyWidget struct {
    widget.BaseWidget
    // 自定义字段
}

func NewMyWidget() *MyWidget {
    w := &MyWidget{}
    w.ExtendBaseWidget(w)  // 必须调用，设置内部 impl 指针
    return w
}

func (w *MyWidget) CreateRenderer() fyne.WidgetRenderer {
    r := &myWidgetRenderer{widget: w}
    return r
}

type myWidgetRenderer struct {
    widget *MyWidget
    // 缓存的绘制对象
}

func (r *myWidgetRenderer) Destroy() {}
func (r *myWidgetRenderer) Layout(size fyne.Size) { /* 排列子对象 */ }
func (r *myWidgetRenderer) MinSize() fyne.Size { return fyne.NewSize(100, 30) }
func (r *myWidgetRenderer) Objects() []fyne.CanvasObject { return []fyne.CanvasObject{...} }
func (r *myWidgetRenderer) Refresh() { /* 更新绘制状态 */ }
```

**DisableableWidget** 继承自 BaseWidget，添加了 `Enable()`, `Disable()`, `Disabled()` 方法。

### `widget.NewSimpleRenderer(object)` 辅助函数

当 Widget 只有一个 CanvasObject 时，可以用这个快捷函数创建 Renderer：

```go
func (w *MyWidget) CreateRenderer() fyne.WidgetRenderer {
    bg := canvas.NewRectangle(color.Black)
    return widget.NewSimpleRenderer(bg)
}
```

## 3. Container 构造速查

容器构造使用 `fyne.io/fyne/v2/container` 包：

| 构造函数 | 布局行为 |
|---------|---------|
| `container.NewVBox(objs...)` | 垂直排列，子元素取 MinSize 高度 |
| `container.NewHBox(objs...)` | 水平排列，子元素取 MinSize 宽度 |
| `container.NewBorder(top, bottom, left, right, center...)` | 四边-中心布局 |
| `container.NewGridWithColumns(cols, objs...)` | 按列数网格排列 |
| `container.NewGridWithRows(rows, objs...)` | 按行数网格排列 |
| `container.NewGridWrap(size, objs...)` | 固定大小网格自动换行 |
| `container.NewAdaptiveGrid(rowcols, objs...)` | 自适应网格（移动端响应） |
| `container.NewStack(objs...)` | 堆叠（后添加的在上面） |
| `container.NewCenter(objs...)` | 居中 |
| `container.NewPadded(objs...)` | 带标准 padding |
| `container.NewWithoutLayout(objs...)` | 无布局（需手动 Move/Resize） |

特殊容器：
| 构造函数 | 说明 |
|---------|------|
| `container.NewHSplit(left, right)` | 水平分割面板（可拖动调整） |
| `container.NewVSplit(top, bottom)` | 垂直分割面板（可拖动调整） |
| `container.NewAppTabs(items...)` | 应用级标签页 |
| `container.NewDocTabs(items...)` | 文档级标签页 |
| `container.NewScroll(objs...)` | 带滚动条的容器 |
| `container.NewInnerWindow(title, content)` | 内嵌窗口 |

## 4. Canvas 原始图形

位于 `fyne.io/fyne/v2/canvas` 包。这些是基础绘制元素，通常被 WidgetRenderer 使用：

| 类型 | 构造函数 | 说明 |
|------|---------|------|
| `Rectangle` | `canvas.NewRectangle(color)` | 矩形 |
| `Circle` | `canvas.NewCircle(color)` | 圆形 |
| `Line` | `canvas.NewLine(color)` | 线条 |
| `Text` | `canvas.NewText("text", color)` | 文本 |
| `Image` | `canvas.NewImageFromResource(res)` | 图像 |
| `LinearGradient` | `canvas.NewLinearGradient(start, end)` | 线性渐变 |
| `RadialGradient` | `canvas.NewRadialGradient(center, radius)` | 径向渐变 |
| `Raster` | `canvas.NewRaster(width, height, drawFn)` | 自定义像素绘制 |

**注意**：`canvas.Line` 的所有方法定义在指针接收者上，必须使用 `*canvas.Line` 而非 `canvas.Line`。

## 5. 数据绑定 (binding) 类型速查

位于 `fyne.io/fyne/v2/data/binding` 包。

### 基础绑定

```go
s := binding.NewString()             // String
b := binding.NewBool()               // Bool
i := binding.NewInt()                // Int
f := binding.NewFloat()              // Float
u := binding.NewURI()                // URI
a := binding.NewUntyped()            // any 类型

// 外部绑定
binding.BindString(&myStr)           // 绑定到已有变量
binding.BindBool(&myBool)
binding.BindInt(&myInt)
binding.BindFloat(&myFloat)
```

### 集合绑定

```go
binding.NewStringList()              // []string
binding.NewBoolList()                // []bool
binding.NewIntList()                 // []int
binding.NewFloatList()               // []float64
binding.NewStringTree()              // 树结构
binding.NewStringMap()               // map[string]string
```

### Widget 的 WithData 构造函数

```go
widget.NewEntryWithData(binding.String)
widget.NewLabelWithData(binding.String)
widget.NewButtonWithData(name, icon, binding.String, tappedFn)
widget.NewCheckWithData("label", binding.Bool)
widget.NewSliderWithData(min, max, binding.Float)
widget.NewSelectWithData(options, binding.String, changedFn)
widget.NewListWithData(binding.StringList, createFn, updateFn)
```


## 8. 存储与资源

```go
// 存储
type Storage interface {
    RootURI() URI
    Create(name string) (URIWriteCloser, error)
    Open(name string) (URIReadCloser, error)
    Save(name string) (URIWriteCloser, error)
    Remove(name string) error
    List() []string
}

// 资源
type Resource interface {
    Name() string
    Content() []byte
}

// 创建/加载资源
fyne.NewStaticResource(name, content)
fyne.LoadResourceFromPath(path)
fyne.LoadResourceFromURLString(url)
fyne.CacheResourceFromURLString(url) // 带缓存

// 偏好存储 (通过 App.Preferences() 获取)
prefs.Bool(key) / prefs.SetBool(key, val)
prefs.Int(key) / prefs.SetInt(key, val)
prefs.Float(key) / prefs.SetFloat(key, val)
prefs.String(key) / prefs.SetString(key, val)
prefs.BoolList(key) / prefs.StringList(key) / ... // v2.4+
prefs.RemoveValue(key)
```

## 11. 动画

```go
type Animation struct {
    AutoReverse bool
    Curve       AnimationCurve
    Duration    time.Duration
    RepeatCount int
    Tick        func(float32)
}

anim := fyne.NewAnimation(duration, fn)
anim.Curve = fyne.AnimationEaseInOut
anim.Start()
anim.Stop()
```

内置曲线：
- `fyne.AnimationLinear` — 线性
- `fyne.AnimationEaseIn` — 缓入
- `fyne.AnimationEaseOut` — 缓出
- `fyne.AnimationEaseInOut` — 缓入缓出（默认）
