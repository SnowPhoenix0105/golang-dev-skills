# Fyne 测试支持

## 1. 测试包概览

`fyne.io/fyne/v2/test` 包提供完整的内存测试能力，无需真实显示环境即可进行 UI 测试。

```go
import "fyne.io/fyne/v2/test"
```

核心能力：
- 创建内存 Canvas，无需图形驱动
- 模拟用户交互（点击、拖动、滚动、输入）
- 验证 Widget 渲染结果
- 检查焦点状态
- 获取 WidgetRenderer 进行内部检查

## 2. 创建测试 Canvas

```go
func TestMyWidget(t *testing.T) {
    // 创建测试 Canvas
    c := test.NewCanvas()
    // 设置内容
    c.SetContent(myWidget)

    // Canvas 拥有 Scale, Size, Focus, Overlays 等完整实现
    c.Resize(fyne.NewSize(400, 300))
}
```

## 3. 模拟用户交互

### 3.1 点击

```go
// 点击可 Tappable 的对象
btn := widget.NewButton("Test", func() { t.Log("clicked") })
test.Tap(btn)

// 在指定位置点击
test.TapAt(btn, fyne.NewPos(10, 10))

// 点击 Canvas 上某个绝对坐标
test.TapCanvas(canvas, fyne.NewPos(100, 50))

// 双击
test.DoubleTap(btn)

// 右键点击
test.TapSecondary(btn)
test.TapSecondaryAt(btn, fyne.NewPos(5, 5))
```

### 3.2 拖动

```go
// 在 Canvas 绝对位置开始拖动
test.Drag(canvas, fyne.NewPos(50, 50), 30, 10) // 拖动 dx=30, dy=10
```

### 3.3 滚动

```go
// 在 Canvas 绝对位置模拟滚动
test.Scroll(canvas, fyne.NewPos(100, 100), 0, -50) // 向上滚动 50
```

### 3.4 键盘输入

```go
// 聚焦 Focusable 对象后输入文字
entry := widget.NewEntry()
test.Type(entry, "Hello World")  // 先 FocusGained，再逐 rune 输入

// 在 Canvas 级别输入
test.TypeOnCanvas(canvas, "Hello")
```

### 3.5 鼠标移动

```go
// 模拟鼠标移动（触发 Hoverable.MouseIn / MouseMoved / MouseOut）
test.MoveMouse(canvas, fyne.NewPos(100, 100))
```

### 3.6 焦点控制

```go
// 切换到下一个/上一个焦点对象
test.FocusNext(canvas)
test.FocusPrevious(canvas)
```

## 4. 验证渲染结果

### 4.1 通过 WidgetRenderer

```go
func TestLabelText(t *testing.T) {
    label := widget.NewLabel("Hello")
    renderer := test.WidgetRenderer(label)

    // 验证渲染器对象
    objects := renderer.Objects()
    if len(objects) != 1 {
        t.Fatal("expected 1 render object")
    }

    text := objects[0].(*canvas.Text)
    if text.Text != "Hello" {
        t.Errorf("expected 'Hello', got '%s'", text.Text)
    }
}
```

### 4.2 使用 RenderToMarkup（v2.6.0+）

```go
func TestMarkupOutput(t *testing.T) {
    btn := widget.NewButton("Click", func() {})
    markup := test.RenderObjectToMarkup(btn)
    // markup 是可读的结构化文本，可用于快照测试
    if !strings.Contains(markup, "Click") {
        t.Error("button text not found in markup")
    }
}
```

### 4.3 Canvas 级别快照

```go
snapshot := test.RenderToMarkup(canvas)
```

## 5. 获取布局后的对象

// LaidOutObjects 返回经过 Layout 处理后的所有 CanvasObject

## 6. 完整测试示例

### 6.1 Widget 功能测试

```go
func TestButton_Tap(t *testing.T) {
    clicked := false
    btn := widget.NewButton("Test", func() {
        clicked = true
    })

    test.Tap(btn)

    if !clicked {
        t.Error("button should have been clicked")
    }
}
```

### 6.2 Widget 渲染测试

```go
func TestButton_Renderer(t *testing.T) {
    btn := widget.NewButton("Submit", nil)
    r := test.WidgetRenderer(btn)

    // 验证有 2 个渲染对象：背景 + 标签
    if len(r.Objects()) != 2 {
        t.Fatalf("expected 2 objects, got %d", len(r.Objects()))
    }

    // 验证最小尺寸
    minSize := r.MinSize()
    if minSize.Width <= 0 || minSize.Height <= 0 {
        t.Error("MinSize should be positive")
    }
}
```

### 6.3 自定义 Widget 测试

```go
func TestMyWidget_Layout(t *testing.T) {
    w := NewMyWidget("test", nil)

    c := test.NewCanvas()
    c.SetContent(w)
    c.Resize(fyne.NewSize(200, 100))

    r := test.WidgetRenderer(w)
    objects := r.Objects()

    // 验证背景充满整个区域
    bg := objects[0] // 假设背景是第一个对象
    if bg.Size().Width != 200 || bg.Size().Height != 100 {
        t.Error("background should fill widget bounds")
    }
}
```

### 6.4 数据绑定测试

```go
func TestEntryWithData(t *testing.T) {
    data := binding.NewString()
    data.Set("initial")

    entry := widget.NewEntryWithData(data)

    test.Type(entry, "new text")

    got, _ := data.Get()
    if got != "new text" {
        t.Errorf("expected 'new text', got '%s'", got)
    }
}
```

### 6.5 焦点测试

```go
func TestFocusChain(t *testing.T) {
    e1 := widget.NewEntry()
    e2 := widget.NewEntry()
    c := test.NewCanvas()
    c.SetContent(container.NewVBox(e1, e2))

    c.FocusNext()           // 聚焦第一个
    focus1 := c.Focused()
    c.FocusNext()           // 切换到第二个
    focus2 := c.Focused()

    if focus1 == focus2 {
        t.Error("focus should have moved to different widget")
    }
}
```

## 7. 创建测试辅助 App

```go
func TestWithApp(t *testing.T) {
    a := test.NewApp()
    w := a.NewWindow("Test")

    // 模拟点击
    btn := widget.NewButton("Test", func() { t.Log("clicked") })
    w.SetContent(btn)
    test.Tap(btn)
}
```

`test.NewApp()` 返回一个内存中的 App 实现，包含完整的 Preferences、Storage、Clipboard 等实现。

## 9. CI 测试

Fyne 支持通过 `ci` tag 在 CI 环境中运行测试（无需图形驱动）：

```bash
go test -tags ci ./...
```

## 10. 测试最佳实践

1. **使用 test.NewCanvas()** 而非真实窗口，测试更快、不需要图形环境
2. **通过 test.WidgetRenderer() 验证渲染**，检查 Objects() 的内容和布局
3. **使用 test.Tap / test.Type 模拟交互**，验证回调是否正确触发
4. **测试 MinSize** 确保 Widget 在不同布局中都能正确显示
5. **测试布局行为**，将 Widget 放入容器中，验证 Layout 后的位置和大小
6. **不要在测试中使用 time.Sleep**，所有操作是同步的
7. **为自定义 Widget 编写 Layout 测试**，验证不同尺寸下的排列行为
